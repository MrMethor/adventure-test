A basic text-based adventure test.
Based on old-school text based adventure games such as Zork or Adventure.

It's more of a curiosity.

Available commands:

North                \
South                 \
West                   \
East                    - Movement
Nortwest               /
Northeast             /
Southwest            /
Southeast           /
take / grab / get     - Take something from the room you are in to your inventory
drop / put            - Put something to the room you are in from yout inventory
read                  - Read an item
inventory             - Shows the inventory
